# Healthcare Analytics in Power BI — Case Study

An end-to-end Power BI case study analyzing hospital efficiency data for **HealthStat**, a fictional healthcare consulting company. The project investigates factors affecting patient **Length of Stay (LOS)** and **hospital costs**, benchmarks hospital performance, and delivers a sophisticated interactive dashboard to communicate insights to the HealthStat executive team.


## Project Overview

Improving efficiency while maintaining high-quality patient care is a top priority in healthcare. HealthStat needed to understand what attributes drive longer patient stays and higher costs across hospitals. This project tackles that challenge — importing and modeling real-world style data, building DAX-powered measures, benchmarking hospitals against each other, and presenting findings in a polished, branded dashboard.

**Key business questions answered:**
- What factors are driving longer patient Length of Stay (LOS)?
- How do hospital costs compare across facilities?
- Which hospitals are outliers in efficiency?
- What is the impact of surgical program size on performance?
- What are the root causes behind inefficiencies?


##  What's Inside

### Healthcare Analysis with Power BI
- Imported and connected the healthcare dataset into a Power BI data model
- Learned about **healthcare efficiency measures** and what drives LOS and cost
- Performed an initial **hospitals overview** and data quality check
- Analyzed **total discharges** and **length of stay** across hospitals
- Built DAX **calculated measures and columns** for exploratory analytics
- Conducted an **initial comparison of hospitals** to set the baseline

### Analyzing Hospital Performance and Benchmarking
- Performed **cost analysis by hospital** to identify high-cost facilities
- Calculated **relative differences** between hospitals for fair benchmarking
- **Highlighted outliers** deviating significantly from the norm
- Investigated **factors impacting efficiency** across the dataset
- Assessed the **impact of surgical program size** on hospital performance
- Conducted **root cause investigation** into inefficiencies

### Creating a Dynamic Dashboard
- Built a cohesive, interactive final dashboard with **HealthStat branding**
- Created an **LOS overview** and **cost comparison overview** report pages
- Built individual **hospital profile** pages for drill-down analysis
- Optimized **user-friendly navigation** across report pages
- Added a **dynamic title** that updates based on user selections
- Brought it all together into a polished executive-ready report
